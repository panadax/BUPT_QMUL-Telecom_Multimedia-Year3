这是2018-2019年徐鹏教授在北京邮电大学国际学院开设的专业选修课《移动互联网》课程。

本课程的考核方式是要求学生撰写2篇2000字的小报告和1篇5000的大报告。

小报告和大报告的选题均有两种模式：

•选题制：将自己拟定的题目发给教师（先提交250字左右报告内容说明），教师同意后开工（期中之前提交选题）

•出题制：教师给出候选题目。如果没有自行选题，则必须完成这两个题目
